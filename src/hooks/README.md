# Custom React Hooks

Esta carpeta contendrá hooks personalizados para encapsular lógica reutilizable:

- `useLocalStorage.js` - Hook para localStorage
- `useCurrency.js` - Hook para conversión de monedas
- `useCategories.js` - Hook para gestión de categorías
- `useTransactions.js` - Hook para transacciones
- `useDebts.js` - Hook para deudas
- `useSavings.js` - Hook para ahorros
- `useInvestments.js` - Hook para inversiones
- `useAnalytics.js` - Hook para análisis (M8, M9)

Los hooks simplifican el uso del estado global en los componentes.
