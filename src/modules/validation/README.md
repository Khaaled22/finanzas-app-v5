# Módulo de Validación

Esta carpeta contendrá utilidades para validar datos:

- `validators.js` - Funciones de validación comunes
- `schemas.js` - Esquemas de validación para formularios

Ejemplos de validaciones:
- Validar que un monto sea positivo
- Validar formato de fecha
- Validar que campos obligatorios no estén vacíos
- Validar rangos de valores

Puedes usar librerías como Yup o Zod en el futuro si es necesario.
