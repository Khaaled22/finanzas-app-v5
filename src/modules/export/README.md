# Módulo de Exportación

Esta carpeta contendrá utilidades para exportar datos:

- `ExcelExporter.js` - Exportación a Excel (usando SheetJS)
- `PDFExporter.js` - Exportación a PDF (M11, fase 3)
- `CSVExporter.js` - Exportación a CSV

Para Excel, instalar:
```bash
npm install xlsx
```

Para PDF (fase 3):
```bash
npm install jspdf html2canvas
```
