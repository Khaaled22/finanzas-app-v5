# Componentes de Gráficos

Esta carpeta contendrá componentes para visualización de datos con Chart.js:

- `ChartWrapper.jsx` - HOC para configuración común
- `BarChart.jsx` - Gráficos de barras
- `LineChart.jsx` - Gráficos de líneas
- `DoughnutChart.jsx` - Gráficos de dona
- `ProjectionChart.jsx` - Gráficos de proyección

**Pendiente:** Implementar M1 (Gráficos en Dashboard)

Primero instalar dependencias:
```bash
npm install chart.js react-chartjs-2
```
