# Componentes Comunes

Esta carpeta contendrá componentes UI reutilizables como:

- `Button.jsx` - Botones estilizados
- `Card.jsx` - Contenedores tipo tarjeta
- `Modal.jsx` - Ventanas modales
- `Input.jsx` - Campos de entrada
- `Select.jsx` - Selectores
- `Badge.jsx` - Etiquetas
- `ProgressBar.jsx` - Barras de progreso
- `MetricCard.jsx` - Tarjetas para métricas

Estos componentes se crearán conforme se implementen las mejoras M1-M9.
