# Componentes de Formularios

Esta carpeta contendrá formularios específicos para cada entidad:

- `TransactionForm.jsx` - Formulario de transacciones (M4)
- `DebtForm.jsx` - Formulario de deudas (M5)
- `SavingsGoalForm.jsx` - Formulario de objetivos de ahorro (M6)
- `InvestmentForm.jsx` - Formulario de inversiones

Estos formularios incluirán validación y manejo de estado.
